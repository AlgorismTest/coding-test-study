### 문제 이해하기
DP의 개념 관련 문제

### 문제 접근 방법
뒤 요소가 해당 요소보다 작고 dp값도 해당 인덱스의 dp값 보다 작으면 dp값 + 1 

### 구현 배경 지식
다이내믹 프로그래밍 적용하기

### 해결하지 못한 이유

### 문제를 해결한 코드
```java
if (arr[i]<arr[j] && dp[i]<=dp[j]){
    dp[i] = dp[j]+1;
}
```

### 문제를 해결한 방법
dp값이 나보다 크거나 같고 arr값이 나보다 크면 dp값 + 1

arr : 30 10 20 20 10 20 
dp  :  1  2  2  2  3  2

<br>
만약에 arr값이 나와 동일하면 dp값은 나와 동일하게 하기
